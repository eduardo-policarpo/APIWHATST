## WhatsApp Api


- Javascript ES6
- NodeJS
- Restfull

## Features

|                                                            |     |
| ---------------------------------------------------------- | --- |
| Structure                                                  | ✔ |
| Multiple Sessions                                          | ✔ |
| Send **text, image, video and docs**                       | ✔ |
| Get **contacts list**                                      | ✔ |
| Get **groups list**                                        | ✔ |
| Receive messages                                           | ✔ |
| Open/Close Session                                         | ✔ |
| Webhook                                                    | ✔ |

## Installation

Install the dependencies and start the server.

```sh
yarn install
//or
npm install
```

## Run Server
```sh
yarn dev
```

## Build Server
```sh
yarn build
```
